🚀 Orbit — macOS App Switcher
Orbit is a lightweight, modern macOS app switcher. It provides a smooth, visually clean way to switch between running applications using a floating overlay and global keyboard shortcuts.

⚠️ macOS Permissions
Orbit requires the following permissions to function correctly:
Accessibility
Needed to activate and switch between applications.
Input Monitoring (depending on system behavior)
Required for global keyboard shortcut detection.
👉 Enable from:
System Settings → Privacy & Security → Accessibility / Input Monitoring

▶️ How It Works
Hold Option (⌥) → Orbit overlay appears
Press Tab → move forward through apps
Release Option → selected app becomes active
